multiview


This page is a multipage inside a fullwidth panel and wrapped by a wrapper page. It contains 3 subpages



Setup

  <!-- wrapper page -->
  <div data-role="page" id="type-home">
	(<!-- global header -->)

    <!-- fullwidth panel -->
    <div data-role="panel" data-panel="fullwidth" data-id="thisPanelsID">

      <!-- regular JQM pages (nested!)>

    </div>
	(<!-- global footer -->)
  </div>

Info

Don't forget to specify data-target="panelID" on all links inside the panel! Be careful not to target to data-panel, which descripes the panel type, like "popover". Since there can be more than one popover on page, the data-target always needs to point to the data-id of the respective panel


One nested page needs data-show="first". This page will be shown first.


By default JQM creates a single history entry for the whole page including all nested pages. You can specify a history for the multipage, which supercedes JQM history by setting data-hash="history" on the panel - not the page! Navigate some pages, click the browser back-button - you will now go back on the panel until the first page labelled with data-show="first". Now clicking the browser back button will take you to the previously visited webpage as the panel-history.stack = 0 and therefore the JQM history takes over.
